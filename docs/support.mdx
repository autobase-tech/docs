# Support

We provide personalized support and expert assistance, so you can focus on building your project with confidence, knowing that a reliable partner is always available when you need help.

:::tip support mail
For support inquiries, please contact us at <a href="mailto:info@autobase.tech">info@autobase.tech</a>
:::

Each Autobase package includes DBA as a Service (DBAaaS)—giving you direct access to a dedicated PostgreSQL expert (DBA).
You’ll also gain access to a private Slack channel for real-time communication and support.
